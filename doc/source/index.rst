Fyplot Documentation
====================

Fyplot is a wrapper over pyqtgraph aiming at making dynamic plotting trivial
(at the expense of configurability).

Installation
------------

.. sourcecode:: none

   # pip install fyplot

After install you may run the demo:

.. sourcecode:: none

   # fyplot_demo

Source
------
