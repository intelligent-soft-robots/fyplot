Fyplot Documentation
====================

Fyplot is a wrapper over pyqtgraph aiming at making dynamic plotting trivial
(at the expense of configurability).

Installation
------------

.. sourcecode:: none

   # pip install fyplot

After install you may run the demos:

.. sourcecode:: none

   # fyplot_demo

.. sourcecode:: none

   # fyplot_demo2

   
Source
------

 * https://github.com/intelligent-soft-robots/fyplot/
